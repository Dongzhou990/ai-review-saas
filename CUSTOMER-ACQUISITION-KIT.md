## 【第一部分】第一批目标客户画像

### 先打谁：皮肤管理工作室

**为什么是他们（不是美甲、不是美发）：**

| 维度 | 皮肤管理工作室 | 美甲美睫 | 美容美发 |
|------|-------------|---------|---------|
| 客单价 | ¥300-3000 | ¥100-500 | ¥100-2000 |
| 新客看评价比例 | **极高（80%+）** | 高 | 中 |
| 老板谁在管 | **通常是女老板自己管** | 老板/店长 | 老板/合伙人 |
| 决策速度 | **快（当场能拍板）** | 快 | 中-慢 |
| 对差评敏感度 | **极度敏感** | 比较敏感 | 中等 |
| 技术门槛 | 中高（专业设备+技术） | 中 | 中低 |
| 开店数量 | 每条街3-5家，竞争激烈 | 更多 | 更多 |

> **结论**：皮肤管理工作室 = 最高客单价 × 最依赖评价 × 老板自己决策 × 差评焦虑最强。
>
> 一个差评劝退的不是一单，是一个可能来 10 次、客单价 ¥1000+ 的长期客户。这个账老板算得过来。

### 皮肤管理老板最痛的 3 个问题

| 排名 | 痛点 | 老板内心独白 |
|------|------|------------|
| 1 | **一条差评劝退几十个新客** | "我就想不通，明明做了那么久，客人写个差评说我技术不行 / 态度不好 / 没效果，新客一看就不敢预约了" |
| 2 | **回了差评客人还是不满意** | "我回了'抱歉给您不好体验'，结果客人追加评论说'老板就会说客套话'。自己写又怕写不好，越写越心虚" |
| 3 | **不知道店里到底哪里出问题了** | "有时候自己也迷糊——是最近新来那个美容师技术不行？还是产品换了一批客人不适应？看不到规律" |

### 我们卖的结果（不是功能）

❌ "我们的 AI 能生成差评回复"
✅ "你店里的差评，以后每条都回得让客人消气——新客看了评价也敢预约"

❌ "我们有差评归因分析"
✅ "一眼看出最近店里到底是服务有问题，还是技术有问题"

❌ "每周自动生成口碑周报"
✅ "每周花 2 分钟，就知道店的口碑是在变好还是变差"

## 【第二部分】第一批 100 家门店筛选标准

### 城市怎么选

**第一优先**：你自己的城市（或你所在的城市圈）

> 前 10 个客户必须当面聊、当面演示。不要去别的城市找。

**第二优先**（等本地跑通后）：同省的省会 + 经济强市
- 广东：深圳、广州、佛山、东莞
- 浙江：杭州、宁波、温州
- 江苏：南京、苏州、无锡
- 四川：成都
- 其他省会

### 门店怎么选

| 筛选维度 | 入选标准 | 排除标准 |
|----------|---------|----------|
| 行业 | 皮肤管理、轻医美、科技美容 | 传统美容院（纯手工按摩类）、美发为主业 |
| 评分 | **3.5 - 4.2 星** | 4.5星以上（口碑太好不焦虑）、3.0星以下（店本身有大问题） |
| 评价数 | **30 - 200 条** | 10条以下（太新没流量）、500条以上（可能是大连锁） |
| 规模 | 单店 或 2-3家连锁 | 5家以上连锁（决策链太长） |
| 经营时长 | 6个月以上 | 新开3个月内的店（还没稳定） |
| 差评状态 | **有 3 条以上差评未回复或回复敷衍** | 所有差评都认真回了 |
| 客单价 | 人均 ¥200+ | 人均 ¥100 以下 |

### 优先级排序（从高到低）

| 优先级 | 门店特征 | 为什么优先 |
|--------|---------|-----------|
| 🔴 最高 | 3.5-3.8星 + 5条以上差评未回 + 有竞争店在旁边 | 口碑危机 + 竞争焦虑 = 最痛 |
| 🟠 高 | 3.8-4.0星 + 3-5条差评回了但很敷衍 | 想做好但不知道怎么回 |
| 🟡 中 | 4.0-4.2星 + 1-2条差评没回 | 口碑还行，有优化空间 |
| ⚪ 不碰 | 4.5星以上 / 所有差评都高质量回复了 | 暂时不需要你 |

### 前 3 个付费客户最可能长这样

> 1. 开了 1-2 年的皮肤管理工作室
> 2. 老板是 28-40 岁女性，自己管店
> 3. 大众点评 3.7-4.0 星（不好不坏）
> 4. 最近 3 个月有 3-5 条差评没好好回
> 5. 同一条街/商圈还有 3 家以上同类店
> 6. 你联系她的时候，她没有直接拒绝

## 【第五部分】客户获取路径

### 渠道一：大众点评（主力渠道，占比 70%）

#### 怎么找

```
1. 打开大众点评 APP
2. 切换到你的城市
3. 搜索框输入："皮肤管理" / "轻医美" / "科技美容"
4. 排序方式选择"评分从低到高"
5. 开始逐条往下滑
```

#### 看什么信号 → 决定加不加

| 看到这个信号 | 动作 |
|-------------|------|
| 评分 3.5-4.0 + 有差评没回复 | ✅ **立刻记进表，A级** |
| 评分 3.5-4.0 + 有差评但回得很敷衍（"抱歉给您不好体验"类模板） | ✅ 记进表，A级 |
| 差评里客人写了很具体的问题，老板没回 | ✅ 记进表，A级（痛点多，成交率高） |
| 评分 4.0-4.2 + 有 1-2 条差评没回 | ✅ 记进表，B级 |
| 评分 4.5 以上 + 所有差评都认真回了 | ❌ 跳过，暂时不需要 |
| 评分 3.0 以下 + 全是差评 + 店可能快倒闭了 | ❌ 跳过，产品救不了 |
| 连锁品牌（5+ 家店） | ❌ 跳过，决策链太长 |
| 传统美容院，没有皮肤管理项目 | ❌ 跳过，不是目标客户 |

#### 搜什么关键词（按优先级）

```
1. "皮肤管理"            ← 最优先
2. "轻医美"               ← 客单价最高的子类
3. "水光针" / "热玛吉"   ← 具体高客单项目
4. "科技美容"             ← 新兴品类
5. "日式美甲美睫"         ← 第二优先品类
6. "美甲美睫"             ← 竞争最激烈的品类
```


## 【第七部分】7 天试用成交 SOP

### 总览

```
Day 1：破冰 + 演示 → 让对方看到"10 秒出回复"
Day 3：轻跟进 → 问他用得怎么样
Day 7：发价值锚点 → 让他感受"除了回差评，还能看懂问题"
Day 14：催付费
```

### Day 1：破冰 + 让 TA 看到效果

**目标**：让对方亲眼看到 AI 回复质量

**具体动作：**
1. 加微信（用【6.1】话术）
2. 通过后立刻发【6.2】第一句话
3. 对方回复后 → 问他要一条差评，或自己上点评找
4. 10 秒内把 AI 回复发给他看
5. 问一句："姐，这个回复你觉得怎么样？"
6. 如果对方说"不错" → 发 Demo 链接
7. 如果对方说"一般" → 问哪里不好，换一种语气再试

**当天必须交付的小结果**：让对方看到**一条针对自家差评的 AI 回复，而且他觉得"确实比自己写得好"**

### Day 4-5：展示附加价值

**目标**：让他从"回差评好用"升级到"这个工具能帮我管好口碑"

**具体动作（选一个发）：**

**如果你能拿到他店里的差评数据：**
> 姐，帮你看了这周的差评。发现一个规律：
> - 最近 3 条差评里，2 条说的是"做完没效果"
> - 可能最近这批产品客人不太适应
> - 建议这周重点跟美容师沟通一下施术流程
>
> （展示分析能力——这是普通回差评做不到的）

**如果你拿不到数据：**
> 姐，除了回差评，这个工具还能每周自动生成口碑周报。比如：
> - 本周差评主要问题是什么
> - 跟上周比口碑变好了还是变差了
> - 建议下周重点改进什么
>
> 老板花 2 分钟看一眼就够了。要不要试试？

### Day 6-7：第一次付费推进

**目标**：让深度使用的人付费，让犹豫的人至少不下船

**对用得多的人（高频用户）：**
> 姐，这周帮你回了不少差评了。你觉得效果怎么样？
>
> 要不要下个月正式用？¥599/月，少丢一个新客就回本。或者先 ¥199 深度体验两周——¥199 可以抵扣后续月费。

**对用得少的人（低频用户）：**
> 姐，这周好像没怎么用？是太忙了还是觉得不好用？
>
> 如果是太忙——你以后直接把差评发我，我帮你回。
> 如果是觉得不好用——告诉我哪里不好，我改。

**对完全沉默的人：**
> 姐，好几天没联系了。你们店最近口碑怎么样？有什么需要我帮忙的吗？

### 第 8-14 天：催付费 or 放弃

**如果前 7 天有互动，但还没付费：**
- 第 10 天再问一次"姐，用得怎么样？"
- 第 13 天发最终催单："姐，两周试用到期了。你觉得有用我们就继续，没用也没关系"

**如果前 7 天完全没互动（消息都不回）：**
- 第 10 天发最后一条："姐，如果你不感兴趣也没关系，跟我说一声就行，我就不打扰你了 🙏"
- 如果还不回 → 放弃这家，换下一家

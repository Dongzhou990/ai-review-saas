## 阶段1: 首轮回复（秒回，亲切感）

**模板A（通用）**
```
你好呀～你是做什么店的呀？👋
```

**模板B（如果对方提到了具体问题）**
```
看到你说[具体问题]了，我特别懂这种感觉！
你是做什么店的？我帮你看看能不能解决～
```

**模板C（如果对方问价格）**
```
每天3条免费，不用绑卡！先试试再说
你是什么店呀？我发演示给你看看👋
```

## 阶段2: 识别门店 + 痛点

```
原来你是做皮肤的！你这个行业差评我有经验。
你们现在主要在哪个平台比较多评价？美tuan还是点评？
```

**根据行业追问：**
- 美甲美睫店 → "客人最常吐槽的是什么？颜色不准还是等太久？"
- 美容院/皮肤管理 → "客人会吐槽效果不满意还是态度问题？"
- 轻医美 → "客人对恢复期不满意还是价格问题比较多？"

## 三、发布排期表

### 发布频率
- **每天 2 条**（上午 10:00-11:00 一条，晚上 20:00-21:00 一条）
- 周一至周五：痛点 + 方案演示 或 干货 + 方案演示
- 周六：认知升级 + 方案演示
- 周日：复盘 + 准备下周内容

### 发布检查清单
```
□ 标题是否包含痛点/数字/情绪词？
□ 正文前15字是否抓人？
□ 是否有产品截图或模板图片？
□ 结尾是否有明确引导（私信/评论/收藏）？
□ 话题标签是否精准（3-5个）？
□ 评论区首评是否已准备好？
□ 私信脚本是否已准备好？
□ 图片顺序是否正确？
□ 今天这个时间点发布合适吗？
```

## 五、私信承接 SOP

```
[客户私信]
  ↓
① 首轮回复（秒回，亲切）
  "你好呀～你是做什么店的呀？👋"
  ↓
② 识别门店类型 + 痛点
  → 美甲/美容/皮肤管理/轻医美/其他
  → 差评不会回 / 没空回 / 好评太少 / 想提高口碑
  ↓
③ 针对痛点给具体价值
  "你这个行业的差评我经常处理，给你看个例子👇"
  [发对应的产品截图或案例]
  ↓
④ 引导试用
  "每天3条免费，不用绑卡，要不要试试看？
   微信扫这个码就能用 [发二维码/链接]"
  ↓
⑤ 跟进（24h后）
  "试用了吗？有什么问题随时问我～"
  ↓
⑥ 成交引导（3-7天后）
  "用了感觉怎么样？现在升级专业版有优惠"
```

### 异议处理手册

| 异议 | 回复 |
|------|------|
| 太贵了 | "少丢一个新客就回本了。一个美业新客值¥300-3000，你算算🤓。而且可以先用免费版，觉得值再升级" |
| 我怕麻烦 | "不麻烦。打开手机粘贴差评就行，10秒。比你现在憋半小时想回复简单多了" |
| 我试过别的，没用 | "别的工具是模板回复，客人一眼看穿。我们是AI读懂差评内容再回复，每条都不一样。试试看就知道区别👇" |
| 我员工会回 | "员工回的跟老板回的，客人看得出来。而且AI能帮你统一回复质量，不用每次都检查" |
| 我没差评 | "那更好！但差评早晚会来。趁现在把口碑系统建好，差评来了不慌。而且好评邀约功能现在就能用" |
| 我再看看 | "好的不着急～你可以先收藏着，哪天差评来了就知道找谁了。也可以先免费用3条试试，反正不花钱😊" |

## 六、数据追踪看板

### 每条笔记必追踪字段

| 字段 | 说明 |
|------|------|
| 笔记编号 | W1-01, W1-02... |
| 发布时间 | 年月日 时分 |
| 标题 | 完整标题 |
| 内容支柱 | 1-4 |
| 笔记目的 | 涨粉/私信/收藏/成交 |
| 24h点赞 | 发布24小时后的点赞数 |
| 24h收藏 | 发布24小时后的收藏数 |
| 24h评论 | 发布24小时后的评论数 |
| 7天点赞 | 7日累计 |
| 7天收藏 | 7日累计 |
| 7天评论 | 7日累计 |
| 私信数 | 此笔记带来的私信数 |
| 试用数 | 私信转化为试用的人数 |
| 成交数 | 最终付费人数 |
| 内容成本 | 生产这条内容花了多少时间 |

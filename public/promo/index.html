📸 如何使用这些图

  1. 启动本地服务：npm run dev
  2. 打开预览：http://localhost:3000/promo/
  3. 单独打开某张图：http://localhost:3000/promo/01-hero.html
  4. 用浏览器截图工具或 Cmd+Shift+4 截取
  5. 如需改文案/颜色/尺寸，编辑 public/promo/ 下的 HTML 文件

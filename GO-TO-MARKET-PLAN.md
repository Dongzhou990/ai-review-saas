## 【1】成交版本重构 — “我们到底卖什么给谁”

### 1.1 最适合卖给谁

**第一优先：皮肤管理工作室 / 轻医美工作室**

| 维度 | 为什么是他们 |
|------|-------------|
| 客单价 | 单次消费 300-3000 元，一个差评丢的不是一单，是一个长期客户 |
| 决策模式 | 老板通常自己管店，决策快，不需要层层审批 |
| 平台依赖 | 极度依赖大众点评/美团/小红书获客，评价就是命根子 |
| 痛点强度 | 一条差评可能劝退 10+ 个潜在新客 |
| 付费能力 | 月流水 5-30 万，¥599-999/月不是问题 |
| 竞争焦虑 | 同一条街上 5 家皮肤管理店，评分差 0.3 就输了 |

**第二优先：美甲美睫店**

| 维度 | 为什么 |
|------|--------|
| 评价密度大 | 客流量大，评价多，管理成本高 |
| 老板年轻 | 对工具接受度高，手机操作无障碍 |
| 口碑敏感 | 新客几乎 100% 先看评价再预约 |

**第三优先：中高端美容美发店**

| 维度 | 为什么 |
|------|--------|
| 客单价高 | 染烫 500-2000 元，差评代价大 |
| 但决策慢 | 很多是传统老板，需要更多教育 |

## 【4】第一批目标客户名单策略

### 4.1 第一批最该找的 100 家店画像

| 特征 | 标准 |
|------|------|
| 行业 | 皮肤管理 > 轻医美 > 美甲美睫 > 美容美发 |
| 城市 | 一二线城市（北上广深杭成渝）+ 省会城市 |
| 评分 | 大众点评 3.5-4.2 星（太低=店本身有问题，太高=不需要你） |
| 评价数 | 30-200 条（太少=没流量，太多=连锁可能） |
| 差评状态 | **有 3 条以上差评未回复，或回复质量差** |
| 客单价 | 人均消费 ¥200+ |
| 经营状态 | 单店或 2-3 家连锁（非大型连锁品牌） |

### 4.2 客户分 A/B/C 级

| 级别 | 定义 | 判断标准 | 策略 |
|------|------|----------|------|
| **A 级** | 本周最可能付费 | 有 5+ 条未回复差评 + 老板亲自管店 + 已经有微信 | **亲自打电话或到店** |
| **B 级** | 本月可能付费 | 有差评焦虑但还没体验过产品 | **密集跟进，先送试用** |
| **C 级** | 长期培育 | 口碑还行，暂时不痛 | **先加上微信，不发广告** |

### 4.3 如何判断优先级（快速筛选法）

打开大众点评 → 搜索"皮肤管理" → 排序"评分从低到高" → 逐个点进去看：

1. 有差评吗？→ 没有 → 跳过
2. 差评回复了吗？→ 回了且不错 → 跳过
3. 差评没回 / 回了很敷衍 → **这就是 A 级客户，马上加微信**

### 4.4 哪类店最容易成为前 3 个付费客户

**画像：**
- 开了 1-2 年的皮肤管理工作室
- 老板是 28-38 岁女性，自己管店
- 大众点评评分 3.8-4.2（不差但也不拔尖）
- 最近 3 个月有 3-5 条差评没好好回
- 有竞争焦虑（同条街还有 3 家同类店）

**为什么是她？**
- 年轻 → 对工具不排斥
- 自己管 → 决策快
- 评分不够高 → 有改善动力
- 竞争激烈 → 口碑差异化是刚需

> "姐，欢迎加入口碑助手！🎉
>
> 从现在开始，你的大众点评差评我帮你盯着。

>
> 10 秒后你就看到效果了。试试看 👇
> reviewai.chat/dashboard/reviews"

### 6.2 第 1 天：让他"有点值"

**目标**：让他在 3 分钟内用 AI 回完 1 条真实差评

**操作**：
1. 发欢迎话术
2. 如果他没回，2 小时后跟进："姐，试了吗？需要我帮你找一条差评演示吗？"
3. 如果他回了且满意 → "看，就这么简单！以后每天打开就能用"
4. 如果他觉得一般 → 问他哪里不满意，手动帮他改

**交付的小结果**：帮他回了 1 条真实差评，而且他看了觉得"确实比我写得好"

### 6.3 第 3 天：让他看到变化

**目标**：让他产生"这个工具确实有点用"的感觉

**操作**：
> "姐，这两天帮你回了 X 条差评，你看看效果怎么样？有需要调整语气的吗？"

**交付的结果**：差评回复率提升（从"没回"变成"回了"），这是第一个可见变化

### 6.4 第 7 天：让他感受到差异

**目标**：展示分析价值，让他从"好用"变成"离不开"

**操作**：
> "姐，帮你看了这周的差评。汇总一下：
> - 一共 X 条差评
> - 主要问题集中在：XX（比如'做完效果不明显'）
> - 建议这周重点培训一下 XX 方面
>
> 这个分析报告，每周会自动生成。你每周花 2 分钟看一眼就行。"

**交付的结果**：他第一次**不用自己总结**就看清了门店的问题

### 6.5 如果他很忙、不爱学

**不要让他自学。** 美业老板大部分时间都在服务客人，没空研究工具。

**正确做法**：
1. 主动帮他开账号
2. 主动帮他回前 3 条差评
3. 每周主动发周报到他微信
4. 他只需要看结果，不需要操作

**这也是陪跑版 ¥1999/月 的价值所在**——你替他做，他只管看结果。

| 环节 | 方式 | 原因 |
|------|------|------|
| 首次接触 | **人工** | 第一印象必须是真人 |
| 演示 AI 回复 | **半自动** | 工具演示，但人工在旁边解释 |
| 注册引导 | **自动化** | 注册流程已经很顺畅 |
| 第 1-3 天跟进 | **人工** | 关键留存节点 |
| 每周周报 | **自动化生成 + 人工推送** | 内容是 AI 生成的，但推送是人工的（发微信） |
| 付费催单 | **人工** | 成交必须真人谈 |
| 续费提醒 | **半自动** | 系统发提醒，人工确认 |

## 【7】14 天拿试用、30 天拿付费 · 作战计划

### 第一阶段：14 天拿第一批试用（目标：10-20 个试用用户）

#### 第 4-7 天：密集试用 + 收集反馈

| **第 7 天** | 第一次反馈收集 | 问每个试用的人"用得怎么样？有什么建议吗？" | 了解真实使用障碍 |

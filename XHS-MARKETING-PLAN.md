### 1.3 人设定位

|------|------|

### 1.4 目标客户画像
```
姓名：李姐
身份：35岁，开了3年美甲店，员工3-5人
痛点：
- 美团/点评上的差评不知道怎么回
- 每天服务客人没空管线上口碑
- 看到差评心情差，但又不能不回
- 新客越来越少，老客复购也降了
行为：
- 手机刷小红书看同行的店怎么运营
- 搜"差评怎么回""美甲店怎么引流"
- 看到干货会收藏，有用会私信问
```

### 1.5 核心卖点（面向客户的表述）
1. **10秒回差评** — 不用憋半小时
2. **每条回复都不一样** — 不像模板，客人看了消气
3. **AI分析差评原因** — 知道问题出在服务还是技术
4. **每天3条免费** — 不绑卡，觉得有用再付费

### 1.6 差异化定位
| 竞品可能在说 | 我们说的 |
|------|------|
| AI赋能门店运营 | 差评来了，10秒回好 |
| 数字化解决方案 | 手机就能用，1分钟搞定 |
| 提升客户满意度 | 让差评客人消气、放心、愿意再来 |
| 智能客服系统 | 不像机器人在说话，像真人在回复 |

## 二、内容支柱（4大支柱）

### 支柱1：戳门店痛点 🔥
目的：引发共鸣 → 关注/收藏  
频率：每周 3-4 条

典型选题方向：
- 差评没人回，新客划走找别家
- 憋半小时想不出一条回复
- 模板回复被客人一眼看穿
- 好评太少，没人主动写
- 不知道最近差评在说什么问题

### 支柱2：解决方案演示 🎯
目的：展示产品能力 → 私信/试用  
频率：每周 3-4 条

典型选题方向：
- 差评粘贴进来，10秒出回复（截图演示）
- 1分钟看完门店口碑（周报截图）
- 不同差评类型怎么回（分类演示）
- 好评邀约文案一键生成
- AI分析：你的差评到底在说服务不行还是技术不行

### 支柱3：门店干货模板 📋
目的：建立信任 → 收藏 → 长期关注  
频率：每周 2-3 条

典型选题方向：
- 美甲店差评回复模板（5种常见差评）

- 皮肤管理店好评回复模板
- 差评来了先别急的3步处理流程
- 门店老板每周口碑自查清单

### 支柱4：老板认知升级 💡
目的：建立专业形象 → 私信咨询  
频率：每周 2 条

典型选题方向：
- 一条差评到底劝退多少新客（数据）
- 回差评不是道歉，是给新客看你的态度
- 好评不是求来的，是设计出来的
- 门店口碑不是"别出事"，是"出事了有人管"
- 为什么你回的差评没效果（3个常见错误）

## 三、转化路径设计

```
笔记曝光
  ↓
评论互动（评论区引导 + 小号提问）
  ↓
私信咨询（自动化首轮回复）
  ↓
识别门店类型 + 痛点
  ↓
引导试用（每天3条免费，不绑卡）
  ↓
试用满意 → 付费升级（¥199/14天 或 ¥599/月）
```

### 关键转化节点

| 节点 | 目标 | 转化率基准 |
|------|------|------|
| 笔记 → 主页访问 | 让目标客户点进主页 | 5-10% |
| 主页 → 私信 | 让访客主动私信 | 15-30% |
| 私信 → 试用 | 让咨询者开始试用 | 40-60% |
| 试用 → 付费 | 让试用者升级付费 | 10-20% |

## 五、内容风格规范

### 5.1 黄金法则
1. **开头必抓人** — 前15字必须戳中痛点或引发好奇
2. **像老板在聊天** — 用"我"不是"我们"，用"你"不是"您"
3. **每段不超过3行** — 手机屏幕宽度，3行就是极限
4. **结果导向** — 说"好评率↑30%"而不是"提升客户满意度"
5. **有具体数字** — 10秒、3条、200+门店
6. **留钩子** — 每条笔记结尾都引导私信/评论/收藏

### 5.2 禁止事项
- ❌ "AI赋能" "数字化" "解决方案" 这类空词
- ❌ 超过5行的段落
- ❌ 没有emoji分割的长文
- ❌ 只讲功能不讲结果
- ❌ 没有引导互动的结尾
- ❌ 像企业官方号的语气

### 5.3 小红书爆文库
**痛点型标题模板：**
- "[场景]，你还在[错误做法]？"
- "做了[X]年[行业]，最怕的就是[痛点]"
- "[数字]%的[目标客户]都在[痛点]"

**结果型标题模板：**
- "[时间]，我让[指标]从[A]到[B]"
- "用了[X]之后，再也不用[痛点]了"
- "[数字]个[行业]老板都在用这个方法"

**反常识标题模板：**
- "其实你不需要[常见做法]"
- "[常见认知]？错！"
- "为什么我不建议你[常规操作]"

**提问型标题模板：**
- "你知不知道你的店[痛点]？"
- "有没有想过为什么[现象]？"
- "你是不是还在[错误做法]？"

### 5.4 正文结构模板
```
[开头1-2句] 痛点场景或引发好奇的问题
[中间3-5段] 分析问题 + 解决思路 + 方法/模板
[结尾1-2句] 总结 + 引导互动（私信/评论/收藏）
[话题标签] 3-5个精准标签
```
